# ATS Resume Analyzer

An AI-powered **ATS (Applicant Tracking System) Resume Analyzer** that evaluates resumes, matches them with job descriptions, and provides actionable feedback.  
Built using **React** for the frontend and **FastAPI** for the backend, with **Google Gemini** for AI-based evaluation.
